# Ethics And Intended Use

Academic Sludge Line helps organize drafting work. It should not be used to
mass-produce deceptive scholarship, fake citations, fake results, or fake peer
review.

## Acceptable Uses

- Creating a first outline from a verified topic brief.
- Turning notes into a draft that remains visibly provisional.
- Running a critique loop before human review.
- Tracking what evidence is missing.
- Producing reproducible revision histories.

## Unacceptable Uses

- Submitting generated prose as verified work without checking it.
- Inventing citations, datasets, regressions, or findings.
- Hiding AI assistance when disclosure is required.
- Creating papers for spam, credential fraud, or publication gaming.

## Operating Principle

The tool should make uncertainty louder, not quieter. A draft with many TODOs is
healthier than a smooth draft with fake confidence.
